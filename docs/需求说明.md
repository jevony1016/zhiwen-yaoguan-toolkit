# 知闻药观工具箱需求说明

## 1. 产品目标

将“知闻药观”沉淀为临床研究、药物安全与标准术语相关的轻量工具箱。第一版先提供 CTCAE v6.0 检索模块，帮助用户快速检索 CTCAE v6.0 条目，解决以下问题：

- 不知道某个实验室指标异常对应哪个 AE 术语。
- 中文术语和英文 EDC 录入术语难以对应。
- CTCAE Grade 1-5 描述不方便快速查看。
- CTCAE v6.0 定义、导航说明和变更内容不方便快速查看。

第一版聚焦“检索”和“对照展示”，不做自动诊断、不做自动分级。

## 2. 合规边界

小程序名称建议为“知闻药观工具箱”，服务类目为“工具 > 信息查询”，定位为学习、资料检索和工作参考工具。

建议在首页、搜索结果页、详情页展示提示：

> 本工具箱仅用于学习、资料检索与工作参考，不构成医学诊断、治疗建议或临床决策依据。

第一版明确不包含：

- 在线问诊
- 临床诊断
- 治疗建议
- 自动 Grade 判定
- 用药建议
- 用户健康数据存储
- 登录和个人信息采集

## 3. 用户输入

用户可以输入：

- 中文术语：贫血
- 英文术语：Anemia
- MedDRA code：10002272
- SOC：血液和淋巴系统异常
- 实验室指标关键词：血红蛋白、Hgb、ALT、AST
- 较长自然语言描述：患者血红蛋白 7.8 g/dL，需要输血，想查看相关 CTCAE 条目

第一版只做本地关键词匹配，不做大模型推理。

## 4. 搜索结果

搜索结果展示：

- 中文术语
- 英文术语
- MedDRA LLT code
- 中文 SOC
- 英文 SOC

点击搜索结果进入详情页。

## 5. 详情页

详情页展示：

- MedDRA LLT code
- 中文术语
- 英文术语
- 中文 SOC
- 英文 SOC
- 定义
- 导航说明
- CTCAE v6.0 变更
- Grade 1-5 中英文对照

Grade 展示规则：

- 中文在上
- 英文在下
- 不给推荐等级
- 不给最终判断

## 6. 第一版技术方案

- 使用微信小程序原生框架。
- 数据放在 `data/ctcae.js`。
- 搜索逻辑放在 `utils/search.js`。
- 不接入服务器。
- 不联网请求。
- 不保存用户输入。
